# Documentation

## Requirements (Used Versions)
- Node >= 8 (1.16.0)
- npm >= 5 (6.1.0)
- yarn >= 1.16.0 (1.16.0)

## Project Setup
- npm install -g @quasar/cli
- quasar create karakal
- quasar mode add electron
- quasar dev / quasar dev -m electron

## Commands
- development: quasar dev
- building: quasar build
